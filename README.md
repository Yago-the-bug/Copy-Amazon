## 🌟 Descrição
Este projeto é uma **recriação visual da Amazon** usando apenas **HTML e CSS**.  
O objetivo é praticar **layout, design responsivo e componentes de e-commerce**, enquanto ainda estou aprendendo **JavaScript**.

Funcionalidades dinâmicas como **autocomplete na barra de pesquisa**, **scroll automático do carrossel** e interações avançadas de botões **ainda não foram implementadas**, mas a interface visual funciona completamente com CSS.

---

## 🛠 Funcionalidades
- 🔍 **Barra de pesquisa:** visual semelhante à Amazon.  
- 🛒 **Carrossel de produtos:** rolagem horizontal com `overflow-x: auto` e `scroll-snap`.  
- 🏷️ **Cards de produtos:** imagem, título, descrição e botão estilizado.  
- 📱 **Layout responsivo:** utilizando Flexbox e CSS moderno.  
- ✨ **Efeitos visuais:** hover, sombras suaves e cantos arredondados.

---

## 💻 Tecnologias
- **HTML5** – Estrutura semântica.  
- **CSS3** – Layout, cores, hover, grid/flexbox e scroll horizontal.
